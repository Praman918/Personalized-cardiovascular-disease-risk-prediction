# 🫀 Personalized Disease Risk Manager

A **cardiovascular disease risk prediction system** using **Federated Learning** — a privacy-preserving AI technique where multiple hospital nodes train a shared model without ever sharing raw patient data.

---

## 📋 Prerequisites

- **Python 3.10 or newer** — download from [python.org](https://www.python.org/downloads/)
- **pip** (comes bundled with Python)
- A terminal / command prompt

---

## 🚀 Setup (First Time Only)

### 1. Extract the ZIP
Unzip the project anywhere, e.g. `C:\Users\You\Desktop\disease-risk-manager\`

### 2. Open a terminal in the project folder
- Windows: hold **Shift**, right-click the folder → **Open PowerShell window here**
- Or open CMD / PowerShell and `cd` to the folder

### 3. Create a virtual environment
```bash
python -m venv .venv
```

### 4. Activate the virtual environment

**Windows (PowerShell):**
```powershell
.venv\Scripts\Activate.ps1
```
**Windows (CMD):**
```cmd
.venv\Scripts\activate.bat
```
**Mac / Linux:**
```bash
source .venv/bin/activate
```
> You should see `(.venv)` at the start of your prompt.

### 5. Install dependencies
```bash
pip install -r requirements.txt
```
This installs PyTorch, Streamlit, FastAPI, scikit-learn, etc. (~5–10 min first time).

---

## 🏋️ Train the Model (Federated Learning Simulation)

Run this **once** before using the app (or any time you want to retrain):

```bash
python run_simulation.py
```

**What it does (automatically, in order):**
1. Generates 10,000 synthetic patient records with 15 real clinical features
2. Splits them across 3 "hospital" client nodes
3. Runs 10 rounds of Federated Learning — each hospital trains locally, only model weights (never patient data) are shared
4. Saves the final global model → `global_model.pth`

Expected output:
```
Step 1: Generating Data..
[Client 1] Starting on port 8001 | 2,667 patient records
[Client 2] Starting on port 8002 | 2,667 patient records
[Client 3] Starting on port 8003 | 2,666 patient records
── Round 1/10 ──
  ✓ Client 1 (2,667 records) — weights received
  ...
  Global Model → Loss: 0.5xxx | Accuracy: xx.xx%
...
Simulation Complete. Global model saved → global_model.pth
```

## 🖥️ Run the Web App

```bash
python -m streamlit run app.py
```

Then open your browser at: **http://localhost:8501**

### What you'll see:
- A clinical form with **5 sections** — Demographics, Blood Pressure, Lab Results, Lifestyle, Symptoms
- Fill in patient values using numeric inputs and radio buttons, and clear normal/risky ranges!
- Click **"Predict My Cardiovascular Risk"**
- Get:
  - 🟢 / 🟡 / 🔴 **Color-coded risk gauge** (LOW / MODERATE / HIGH)
  - **Elevated risk factor badges** (e.g. Smoker, High LDL, Chest Pain)
  - **Personalized clinical recommendations**
  - **XAI Feature Risk Contribution bars** for all 15 clinical features

## 🔬 Clinical Features Used

| Feature | Description | Normal Range |
|---|---|---|
| Age | Patient age in years | 20–90 |
| Sex | Biological sex | Female / Male |
| BMI | Body Mass Index | 18.5–24.9 |
| Systolic BP | Systolic blood pressure | < 120 mmHg |
| Diastolic BP | Diastolic blood pressure | < 80 mmHg |
| Total Cholesterol | Total blood cholesterol | < 200 mg/dL |
| HDL Cholesterol | "Good" cholesterol | > 60 mg/dL |
| LDL Cholesterol | "Bad" cholesterol | < 100 mg/dL |
| Fasting Blood Glucose | Blood sugar after fasting | 70–99 mg/dL |
| Smoking Status | Current smoker | No |
| Physical Activity | Days/week of exercise | ≥ 5 days |
| Family History | Family history of CVD | No |
| Chest Pain | Chest pain/discomfort | No |
| Shortness of Breath | Dyspnea | No |
| Chronic Fatigue | Persistent unexplained tiredness | No |

## ⚠️ Troubleshooting

| Problem | Solution |
|---|---|
| `ModuleNotFoundError` | Make sure your virtual environment is activated (`(.venv)` in prompt) |
| Port 8501 already in use | Run `python -m streamlit run app.py --server.port 8502` |
| `global_model.pth not found` error in app | Run `python run_simulation.py` first |
| Simulation hangs / clients time out | Kill all Python processes and re-run `python run_simulation.py` |
| Low risk shown for all inputs | Delete `global_model.pth` and retrain with `python run_simulation.py` |
